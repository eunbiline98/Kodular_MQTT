# Kodular MQTT
Tugas Kodular MQTT broker EMQX By Kelas IoT

Untuk tugasnya yaitu : 
Melakukan publish dan subscribe dengan sensor apapun lengkap dengan mobile app nya.

# Device 
- ESP32
- DHT22
- LED

# Tools
- [Kodular](https://kodular.io)
- [Wokwi](https://wokwi.com)
